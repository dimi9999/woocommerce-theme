== Description ==

**Add a rating to your product and view a report!**

*This plugin add a rating star to your product. 
*You can vote products one time in a day.
*Inside Plugin Option you can view votes for all products.
*You can choose color for stars.

https://www.youtube.com/embed/TTsYbmAzFsA

*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at <a href="mailto:report@nsthemes.com">report@nsthemes.com</a>.*

== Installation ==

1. Upload the entire 'ns-product-rating-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Settings' -> 'NS Product Rating'
