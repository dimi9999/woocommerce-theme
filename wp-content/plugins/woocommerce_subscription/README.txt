WooCommerce extension allow clients make payment with Fondy.

== Description ==
This plugin adds Fondy as payment method to your WooCommerce store. Fondy can accept VISA and MasterCard securely, quickly and easily on your store in minutes. Simple, straightforward pricing, top notch fraud intelligence, and 24/7 support.

= Start with Fondy =
[Create Free Fondy Account](https://portal.fondy.eu/mportal/#/account/registration)

= Features =
* Customize your checkout window ([Some styles](http://jsfiddle.net/8h65h91t/))
* Enable/Disable Pop-up mode
* Easy configuration in WooCommerce - only Merchant ID and Secret Key need to be copied from Fondy Merchant Portal

= Allowed currencies =
With us your customers can make purchases in many currencies. Fondy support  EUR, USD, GBP, RUB and UAH.

== Frequently Asked Questions ==
Some answers you can find here [FAQ](https://fondy.eu/faq/)

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After activation, you can set options from `WooCommerce -> Settings -> Payment (tab "Fondy")
